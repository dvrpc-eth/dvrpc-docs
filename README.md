# DVRPC Documentation

Decentralized Verified RPC node for Ethereum.

## Overview

DVRPC serves JSON-RPC with cryptographic proofs. It integrates existing light clients for consensus verification, adds proof generation and P2P layer on top. Pluggable architecture - not tied to any single light client implementation.
